# Required Fonts

Download and install the following fonts before using md2pdf. To install on macOS, double-click each font file and click "Install Font."

## Minion Pro (required for both styles)

Adobe's Minion Pro serif typeface. Used for body text, headings, and code blocks.

Download: https://font.download/font/minion-pro

## Lato (required for Alumni Chapel style)

Google's Lato sans-serif typeface. Used for table headers, H2/H5/H6 headings, footnotes, and captions.

Download: https://fonts.google.com/specimen/Lato

## STIX (required for Alumni Chapel style)

The STIX serif typeface. Used for H1 display headings and blockquote callouts.

Download: https://github.com/stipub/stixfonts
